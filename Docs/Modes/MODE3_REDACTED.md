# Mode 3 — Kinetic Energy Assist

## ⬛ REDACTED — ITAR / INVENTION SECRECY ACT REVIEW PENDING

Mode 3 describes a kinetic energy assist system that may involve technologies subject to:

- **International Traffic in Arms Regulations (ITAR)** — 22 CFR Parts 120-130
- **Invention Secrecy Act of 1951** — 35 U.S.C. §§ 181-188
- **Patent Security Category Review List** — DoD Directive 5535.02

The content of Mode 3 has been withheld from this public release as a precautionary measure. The inventor believes in open-source disclosure but also recognizes the responsibility to ensure that technologies with potential export control or national security implications are handled appropriately.

## What Can Be Said

Mode 3 involves a fourth power routing option that enhances projectile performance using stored recoil energy. It builds on publicly known electromagnetic acceleration principles but applies them in a novel configuration within the REHD architecture.
