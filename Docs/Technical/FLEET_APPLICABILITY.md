# Fleet-Wide Applicability

## Any Vessel With a Recoil-Operated Gun System

REHD is not limited to a single ship class, flight variant, or navy. Any platform mounting a gun with a recoil mechanism can be retrofitted.

## Beyond Mk 45 — Other Gun Systems

The REHD concept applies to any recoil-operated weapon. Future expansion candidates:

- **76mm Oto Melara** (widely deployed on frigates/corvettes worldwide)
- **57mm Bofors Mk 3** (LCS, international frigates)
- **155mm howitzers** (ground forces — M109 Paladin, PzH 2000)
- **120mm tank guns** (M1 Abrams, Leopard 2)
- **105mm aircraft guns** (AC-130 gunship)

Each requires a separate energy budget analysis, but the core concept — harvest recoil KE via linear generator, route to useful destination — is universal.
